**第三阶段再做 Web 控制台**

Web 控制台负责：

- 可选择是1.RD成像还是2.模型推理模式
- 可选择1.自动蛇形patch模型，2.手动操控的无人机模拟飞行模式，支持上下左右wasd（无人机模式实现依然留空）
- 自动蛇形patch模式：start / pause / stop / reset
- 可选择1.HDMI 显示模式，2.png保存模式
- 可自由选择底图：列出可选底图并点击加载
- 有设置页面，可设置一些飞行参数和RD参数，具体逻辑看config文件
- 目前没有实现的模块留空出接口就行
